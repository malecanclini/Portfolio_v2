# Alejandra Canclini — Portfolio

A personal portfolio website showcasing my front-end development projects, built from scratch with HTML, CSS, and Bootstrap.

🌐 **Live site:** [portafolioalecanclini.netlify.app](https://portafolioalecanclini.netlify.app/)

## About

This portfolio was designed and coded by Alejandra Canclini, a front-end developer in training based in Paraguay. It features a clean, responsive layout and serves as a central hub for my web development work, including projects focused on social impact, data visualization, and interactive maps.
